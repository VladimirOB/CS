Шаг 7: Создание виртуальных хостов на Apache

Для начала нам нужно создать каталог webroot 
для домена sedicomm.com:

mkdir -p /var/www/html/sedicomm.com/

Далее мы назначим необходимые разрешения 
для каталога, используя переменную $USER:

chown -R $USER:$USER /var/www/html/sedicomm.com/

Затем назначьте необходимые разрешения 
для каталога webroot для домена:

chmod -R 755 /var/www/html/sedicomm.com

Теперь, используя текстовый редактор, 
создайте пример файла index.html:

nano /var/www/html/sedicomm.com/index.html

<html>
<head>
<title>Welcome to Sedicomm.com</title>
</head>
<body>
<h1>Sedicomm.com</h1>
</body>
</html>

Теперь создайте файл виртуального хоста 
для домена с помощью команды, показанной ниже:

nano /etc/apache2/sites-available/sedicomm.com.conf

Теперь скопируйте и вставьте содержимое ниже, 
замените домен sedicomm.com своим доменом:

<VirtualHost *:80>
ServerAdmin admin@sedicomm.com
ServerName sedicomm.com
ServerAlias www.sedicomm.com
DocumentRoot /var/www/html/sedicomm.com/
ErrorLog ${APACHE_LOG_DIR}/error.log
CustomLog ${APACHE_LOG_DIR}/access.log combined
</VirtualHost>

На этом этапе включите файл виртуального хоста, 
как показано ниже:

a2ensite sedicomm.com.conf

Теперь нужно отключить сайт по умолчанию:

a2dissite 000-default.conf

Чтобы внести изменения, 
перезагрузите веб-сервер Apache.

systemctl restart apache2
